# Qwen Ecosystem - Advanced AI Assistant Framework

Welcome to the Qwen ecosystem! This directory contains an advanced AI assistant framework that integrates capabilities from the Cursor ecosystem and other AI tools into a unified, self-documenting, adaptive system.

## Overview

This system brings together advanced capabilities from multiple AI ecosystems, with a focus on:
- Systematic debugging and problem-solving approaches
- Structured workflows for creative and analytical work
- Ecosystem management and optimization
- Cross-platform coordination and integration
- Self-improvement and continuous learning

## Key Components

### Advanced Skills
Located in `skills/` directory:
- 59 skill directories, including superpowers adapters, ToolUniverse/science skills, plugin/MCP development skills, messaging-access skills, ecosystem-audit skills, and creative/media skills
- Canonical superpowers are authored upstream in `/Users/steven/supremepowers/skills/`, then propagated into Qwen adaptation layers
- Use `QWEN_INVOKE_INDEX_BY_TYPE.md` when you need the roster grouped by skills, agents, scripts, commands, hooks, and external workflow formats

### Agents and Workflows
- 20 Qwen agent directories under `agents/`
- 3 slash-command style workflow docs under `commands/`

### Scripts
Located in `scripts/` directory:
- `update.sh` - Update and synchronize the Qwen ecosystem
- `qwen-sp` - Qwen Superpowers orchestrator
- `qwen-backup` - Backup, verify, list, and restore Qwen ecosystem snapshots

## Getting Started

1. **Read the system definition** to understand the complete framework
2. **Consult the master recall index** to find appropriate capabilities
3. **Use the use cases and triggers** to match your goals to capabilities
4. **Apply the working principles** for consistent behavior

## Philosophy

This system emphasizes:
- **Systematic approaches** over ad-hoc solutions
- **Root cause analysis** rather than symptom fixing
- **Quality over speed** in all operations
- **Continuous improvement** and learning
- **Integration** across AI tools and platforms

## Integration with Other Ecosystems

This system is designed to work alongside Cursor, Claude, and other AI tools, with capabilities that can be adapted and transferred between platforms as needed.

## Maintenance

The system is designed to be self-updating and adaptive. As new capabilities emerge in any ecosystem, they should be integrated following the patterns established in the system definition.
