# Plan document snapshots from local installs

These files preserve **dated filenames** and exact bytes from other trees on disk. Canonical working copies in this repo often use **shorter names** under `docs/plans/` (for example `phase2-implementation.md`).

| Subfolder | Source |
|-----------|--------|
| `gemini-extension/` | `~/.gemini/extensions/supremepower/docs/plans/` |
| `qwen-integration/` | `~/.qwen/integrations/supremepower/docs/plans/` |

Do not edit snapshots for day-to-day work; update the main `docs/plans/*.md` files (or the live install) and refresh snapshots if you need a new frozen copy.
