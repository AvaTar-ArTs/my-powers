# SupremePower Documentation

SupremePower is a multi-platform skills and agent orchestration framework for AI coding assistants. This documentation site is the operational source for installation, architecture, runtime flow behavior, and release governance.

## What this docs site covers

- Platform setup and runtime models (Gemini CLI, Claude Code, Codex, OpenCode)
- Skills and agents architecture
- Hooks, bootstrap, and flow verification expectations
- Ecosystem pathing and host alignment
- Changelog and release-note practices

## Canonical repository

- GitHub: [AvaTar-ArTs/my-powers](https://github.com/AvaTar-ArTs/my-powers)

## Source-of-truth orientation

- Runtime implementation lives in repository roots such as `skills/`, `core/`, `hooks/`, `lib/`, `mcp-server/`, and `commands/`.
- Verification and harnesses live in `tests/`.
- Historical and operational context lives in `docs/`, `CHANGELOG.md`, and `RELEASE-NOTES.md`.
